### Groundwater Recharge Principles
To improve underground water levels, we must manage how rainwater moves across the land.

| Method | Description | Primary Benefit |
| :--- | :--- | :--- |
| **Contour Bunds** | Earth embankments built along elevation lines. | Reduces soil erosion and allows percolation. |
| **Check Dams** | Barriers constructed across flowing streams. | Slows runoff speed for irrigation. |
| **Percolation Tanks** | Man-made basins designed to capture runoff. | Replenishes underground aquifers. |
| **Borewell Recharge** | Directing filtered water into existing wells. | Instantly replenishes deep sources. |
